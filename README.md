# Overview
This is a Streamlit-based chatbot system designed to assist patients preparing for a colonoscopy at Taipei City Hospital Renai Branch. Powered by a Retrieval-Augmented Generation (RAG) pipeline with ChromaDB as the vector store and a large language model as the response engine, this tool provides accessible, real-time answers to common patient questions.

You can view the [live demo here](https://colonoscopyrag.streamlit.app/).

# Use Case

This app is deployed at **Taipei City Hospital Renai Branch** to support patient education during health check-ups involving colonoscopy. It serves as an auxiliary tool to:

* Reduce repetitive staff workload
* Improve patient understanding of pre-/post-exam instructions
* Deliver reliable information in an on-demand, conversational format
